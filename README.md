How to Use the Program
The application is fully interactive and can be controlled with both mouse and keyboard. The following actions are available:
The application is interacted by mouse and keyboard.
1.	When starting the program, three objects (cube, triangle, and sphere) are located on the screen.
2.	To select an object, click the left mouse button, and the objects edges will highlight, meaning that it is selected. This object can be:
a.	Moved using left, right, up, and down arrow keys
b.	Rotated clockwise and counter clockwise using R and T buttons
3.	Camera can be moved pressing and holding left mouse button, and zoomed in or out using mouse scroll.
